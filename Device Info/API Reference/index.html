    API Reference

    This is the API Reference for Device Info module.

    For usage how to use, see the usage page

    Expect several grammar problems in this page, my native language is not English.

    Warning

    Remember that you can only use this module on the client, not the server.

    [module.Enum.PlatformType] module.getDevicePlatform()¶

    Returns the client's Platform type. (Example: Mobile, console, and PC)

    [module.Enum.DeviceType] module.getDeviceType()¶

    Returns the client's device type. (example: Tablet, Phone, PC)

    [module.Enum.InputType module.getInputType()¶

    Returns the client's input type. (example: Keyboard and mouse, gamepad)

    [module.Enum.DeviceOrientation/nil] module.getDeviceOrientation()¶

    Returns the current device orientation if the user is on a mobile device, or else nil.

    Warning

    Remember you should be using this if the device is a mobile device, or else it will return nil.

    [module.Enum.GraphicsQuality] module.getGraphicsQuality()¶

    Returns the current graphics level quality.

    [Vector2] module.getDeviceScreenSize()¶

    Returns the Roblox window size in pixels.

    [RBXScriptSignal] module.inputTypeChanged¶

    Returns a RBXScriptSignal, fires with module.Enum.InputType when the device input type changes.

    [RBXScriptSignal] module.screenSizeChanged¶

    Returns a RBXScriptSignal, fires with the new screen size's Vector2 when the Roblox window size changed in pixels.

    [RBXScriptSignal] module.screenOrientationChanged¶

    Returns a RBXScriptSignal, fires with module.Enum.DeviceOrientation when the mobile device's orientation changed. Will not fire if the device isn't on mobile.

    [RBXScriptSignal] module.graphicsQualityChanged¶

    Returns a RBXScriptSignal, fires with module.Enum.GraphicsQuality.

    [PlatformType] module.Enum.PlatformType¶

    This enum is used for module.getDevicePlatform().

    Contains:

    • module.Enum.PlatformType.Computer - Indicates that the client's platform type is a computer.

    • module.Enum.PlatformType.Console - Indicates that the client's platform type is a console.

    • module.Enum.PlatformType.Mobile - Indicates that the client's platform type is a mobile device.

    [DeviceType] module.Enum.DeviceType¶

    This enum is used for module.getDeviceType().

    Contains:

    • module.Enum.DeviceType.Computer - Indicates that the client's device type is a computer.

    • module.Enum.DeviceType.Console - Indicates that the client's device type is a console.

    • module.Enum.DeviceType.Phone - Indicates that the client's device type is a phone.

    • module.Enum.DeviceType.Tablet - Indicates that the client's device type is a tablet.

    • module.Enum.DeviceType.TouchscreenComputer - Indicates that the client's device type is a touchscreen computer.

    [DeviceOrientation] module.Enum.DeviceOrientation¶

    This enum is used for module.getDeviceOrientation().

    Contains:

    • module.Enum.DeviceOrientation.Landascape - Indicates that the client's device orientation is landascape.

    • module.Enum.DeviceOrientation.Portrait - Indicates that the client's device orientation is portrait.

    [InputType] module.Enum.InputType¶

    This enum is used for module.getInputType().

    Contains:

    • module.Enum.InputType.Touchscreen - Indicates that the client's input type is touchscreen.

    • module.Enum.InputType.KeyboardMouse - Indicates that the client's input type is keyboard with mouse.

    • module.Enum.InputType.Gamepad - Indicates that the client's input type is gamepad.

    • module.Enum.InputType.VR - Indicates that the client's input type is a VR.

    • module.Enum.InputType.Keyboard - Indicates that the client's input type is a keyboard.

    • module.Enum.InputType.Mouse - Indicates that the client's input type is a mouse.

    [GraphicsQuality] module.Enum.GraphicsQuality¶

    This enum is used for module.getGraphicsQuality().

    Contains:

    • module.Enum.GraphicsQuality.Automatic - Indicates that the client's current graphics quality is automatic.

    • module.Enum.GraphicsQuality.Level01 - Indicates that the client's current graphics quality is level 01.

    • module.Enum.GraphicsQuality.Level02 - Indicates that the client's current graphics quality is level 02.

    • module.Enum.GraphicsQuality.Level03 - Indicates that the client's current graphics quality is level 03.

    • module.Enum.GraphicsQuality.Level04 - Indicates that the client's current graphics quality is level 04.

    • module.Enum.GraphicsQuality.Level05 - Indicates that the client's current graphics quality is level 05.

    • module.Enum.GraphicsQuality.Level06 - Indicates that the client's current graphics quality is level 06.

    • module.Enum.GraphicsQuality.Level07 - Indicates that the client's current graphics quality is level 07.

    • module.Enum.GraphicsQuality.Level08 - Indicates that the client's current graphics quality is level 08.

    • module.Enum.GraphicsQuality.Level09 - Indicates that the client's current graphics quality is level 09.

    • module.Enum.GraphicsQuality.Level10 - Indicates that the client's current graphics quality is level 10.
